## Generate QR Code

You can generate a QR code for the landing page using any QR code generator:

1. Visit [QR Code Generator](https://www.qr-code-generator.com/)
2. Enter URL: `https://aem90.github.io/babathings-central-styling/coffee.html`
3. Download and print the QR code
4. Place it near your coffee grinder!

## Quick Setup for Coffee Station

1. Print the QR code
2. Laminate it (optional, for durability)
3. Place it near your coffee grinder or brewing station
4. Scan to instantly log your brews!
